# CUnit

Copy of CUnit unit testing in C Language (http://cunit.sourceforge.net/) version 2.1.2

## ready

```
apt-get install gcc cmake -y
```

## install

```
./build.sh
```

## example

```
cd ./example
./ctest_sdv
```
